Usage

Installation

requirement:
sudo apt-get install portaudio19-dev
Clone the source code and install require packages
git clone https://github.com/NTUEE-ESLab/2017Fall-IntelligentAssistant.git
cd 2017Fall-IntelligentAssistant
pip install -r requirements.txt
Install Tizonia
curl -kL https://github.com/tizonia/tizonia-openmax-il/raw/master/tools/install.sh | bash
Download and compile your DHT11 driver module
cd dht11/dht11
make
Arduino setting
1. Use Arduino IDE and deploy the code 'BLEreceiver.ino' to your Arduino
2. Pin setting for our fan :
　Arduino < --- > L298N
          3                         enA
          5                         in1
          6                         in2
(for free to change by your self)
3. You should turn on Arduino before run the code
4. NOTE : you should change the address in BLE.py if you use your BLE shield
NOTE :
1. you need to set up the mic/speaker device by your self

Run

Simply run rpihome.py

If you want to deploy this service to your Rpi, just edit the .bashrc file

Configuration

You can edit the DEBUG variable in rpihome.py to 1, if you do so, you can type the command via terminal instead of saying it.
